

# Restaurant Website 

## Overview  
This repository contains a responsive restaurant website built with **HTML**, **CSS**, and  **JavaScript**.  
It features multiple pages: Home, Menu, Recipes, Delivery, About, game, and a shared Navbar component across all pages.  

## Pages Structure  
- **Home** – Welcome landing page with a hero section and food categories.  
- **Menu** – Displays sections for Foods, Sandwiches, Drinks, and Extras with cards.  
- **Recipes** – A grid of recipe cards, with a modal for details when clicking a card.  
- **Delivery** – Highlights the delivery service information and link to order.  
- **About** – Contact form for users to send messages, plus brand identity.  
- **Navbar** – Common navigation bar loaded via `nav.js`, used on all pages.  

## Features  
- Fully responsive design across mobile (≤ 600px), tablet (≤ 992px), and desktop screens.  
- Clean and consistent styling for a modern look with brand colors.  
- Shared components (Navbar) to avoid duplication across pages.  
- Simple contact form and redirect/flow for ordering or interaction.  

## Technologies Used  
- HTML
- CSS
- JavaScript
- Font Awesome for icons
